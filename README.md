

# Wedding Invitation / Script Undangan Digital
Web Undangan Pernikahan / Wedding Invitation 


## 📦 Documentation
- Upload Repository ini ke hosting / local host kalian
- Buat Database Msyql dan Import SQL febrizip.sql
- 2 Halaman (Dashboard & Halaman Undangan)
- Halaman Dashboard untuk setting komentar, dll.
- Halaman Undangan adalah halaman utama.
> Undangan ini Full Dinamis
> Splash screen, Amplop, dan halaman undangan, Music, Dark / Light Mode, Ucapan, Map, dll
> hanya menggunakan HTML, CSS, dan JavaScript biasa.
  
## ⚙️ Tech stack

- Bootstrap 5.3.8
- AOS 2.3.4
- Fontawesome 7.1.0
- Canvas Confetti 1.9.3
- Google Fonts
- Vanilla JS
- Native
